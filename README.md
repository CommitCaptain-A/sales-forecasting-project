# Sales Forecasting for Retail

## 📌 Project Overview
This project predicts retail sales using historical data and machine learning techniques.

## 📊 Dataset
- Walmart Sales Dataset
- Includes features like Date, Store, Weekly Sales, Holiday Flag, CPI, and Unemployment

## ⚙️ Methodology
- Data preprocessing
- Feature engineering (year, month, week, lag feature)
- Model: Random Forest Regressor

## 📈 Results
- Model evaluated using RMSE
- Successfully captures sales trends

## 📌 Key Insights
- Store is the most important feature
- Sales show seasonal patterns
- Lag feature helps capture past trends

## 🚀 Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
